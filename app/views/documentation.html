Documentação do Projeto

Foram utilizadas as seguintes tecnologias:

  • NPM
  • Bower
  • Grunt
  • AngularJs
  • Bootstrap
  • Karma
  • Jasmine
Além dessas utilizei o Yeoman para gerar o código padronizado.

Instruções de Instalação

Após baixar o projeto executar: 

npm install

bower install

npm install karma-jasmine --save-dev
Para rodar: 
grunt serve
Para rodar teste: 
grunt test

Decisões do projeto

Criei a pasta service e fiz o serviço que obtém o jSon gerado no mocky. Criei um html para cadastrar, editar e visualizar contatos. Criei um único controle onde realizo as operações de pegar a lista do json através do service e setar em uma lista do controller, adiciono contato a lista, edito contato , removo contato e visualizo contato.
